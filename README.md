# 🧠 PsychHub

PsychHub is a psychology student productivity platform designed to help organise research papers, notes, and study tasks in one place.

The goal of PsychHub is to create a central hub for psychology students to manage their academic work more efficiently.

## ✨ Features

### 📚 Research Papers
- Add psychology research papers
- Organise papers by topic
- Search through saved papers
- View individual paper pages

### 📝 Study Notes
- Create notes for individual papers
- Save notes locally
- Keep important research information together

### 📅 Study Planner
- Add study tasks
- Organise revision activities
- Track academic goals

## 🛠️ Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS
- Git & GitHub
